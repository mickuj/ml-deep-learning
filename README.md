# ML & Deep Learning Projects

Collection of machine learning and deep learning projects focused on regression, computer vision, and sequence modelling.

The repository contains three end-to-end projects implemented in Python using scikit-learn, PyTorch, TensorFlow/Keras, and XGBoost.

---

# Projects

## 1. Used Cars Price Prediction

Regression project focused on predicting used car prices using classical machine learning models and feature engineering.

### Features

* preprocessing pipelines,
* feature selection,
* hyperparameter tuning,
* model comparison,
* feature importance analysis.

### Models

* Linear Regression,
* Random Forest,
* XGBoost,
* Gradient Boosting,
* MLP Regressor.

**Tech stack:** scikit-learn, XGBoost, pandas, NumPy

---

## 2. Musical Instrument Classification

Computer vision project for musical instrument recognition from images using CNNs and transfer learning.

### Features

* custom CNNs,
* data augmentation,
* pretrained models,
* confusion matrices,
* training visualizations.

### Models

* Custom CNN,
* ResNet,
* EfficientNet,
* DenseNet,
* MobileNet.

**Tech stack:** PyTorch, torchvision, scikit-learn

Dataset sourced from Kaggle.

---

## 3. Music Generation with RNNs & Transformers

Sequence modelling project based on MIDI piano recordings and next-note prediction.

### Features

* MIDI preprocessing,
* symbolic music tokenization,
* recurrent neural networks,
* Transformer encoder,
* generated sequence evaluation.

### Models

* SimpleRNN,
* LSTM,
* GRU,
* BiLSTM,
* Transformer Encoder.

**Tech stack:** TensorFlow/Keras, pretty_midi, NumPy

Dataset based on GiantMIDI-Piano (Kaggle subset).
